ColorScope Help

Usage

Defining the simulation region

To define the simulation region, activate ColorScope and position/resize its window until the content you wish to examine is visible through the frame.

Changing simulation mode

To change the simulation mode, activate the popup menu in the top-right of the ColorScope window and click on the new mode. The selected mode will be applied immediately, and ColorScope always starts up with the previously selected mode active.

The available modes are:

  • Deuteranopia (red-green)
  • Protanopia (red-green)
  • Tritanopia (blue-yellow)
  • Achromatopsia (limited color)
  • No Color Correction

Saving a screenshot

To save a screenshot of the simulation region, click the camera icon in the top-left of the ColorScope window. A dialog box will appear prompting you to select the output format and location.
